AI-Based Identity Verification and Liveness Detection System

This AI-powered identity verification system compares a person’s ID card photo with their live video to confirm if both represent the same individual. Using advanced face recognition algorithms, it calculates a similarity score to determine identity match accuracy. Additionally, the system performs liveness detection to check if the person in the video is genuinely live — not a static image, pre-recorded video, cartoon, or AI-generated deepfake. It generates a liveness score to prevent spoofing attacks. This technology helps ensure secure, real-time identity verification in applications like online KYC, remote onboarding, and fraud prevention.

AI Voice Agent for Smart Appointment Booking

This AI voice agent automates appointment booking through natural conversations. Built for salon services, it allows users to schedule appointments via phone by interacting with the AI. The system handles real-time speech, checks available time slots, applies custom rules (like stylist selection, working hours, and break times), and confirms bookings instantly. It improves customer experience, reduces manual work, and ensures accurate, 24/7 booking — making it adaptable for any service-based business.

Secure Telegram VIP Bot for Protected Image Access

This Telegram bot provides secure, VIP-only access to premium images via time-limited links. Registered users can receive private photo links posted in a VIP channel. Each link opens the image in a browser but becomes invalid if shared outside the channel, preventing unauthorized access or forwarding. The bot ensures only verified users can view the content, making it ideal for exclusive media sharing, private communities, or subscription-based services.
